# IngressoTop — versão sem eventos

Esta versão foi preparada para você aprender a cadastrar o primeiro evento manualmente.

## Como adicionar seu primeiro evento

1. Abra `index.html`.
2. Pesquise por `ADICIONE SEU PRIMEIRO EVENTO AQUI`.
3. Copie o bloco `<article class="event-card">...</article>` que está dentro do comentário.
4. Cole fora do comentário, dentro de `<div class="events-grid" id="eventsGrid">`.
5. Altere nome, cidade, modalidade, data, descrição e link.
6. Salve e publique:

```bash
git add .
git commit -m "Adiciona primeiro evento"
git push
```

A Vercel atualizará automaticamente.
